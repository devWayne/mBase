# mBase
> Ns css is used in the movie business department of dianping.com

Powered by Less & Postcss

# Usage

**Install with Bower:**

```shell
$ bower install --save mb.css

$ bower install --save mb.lite.css
```

# Build From Source

```shell
$ git clone https://github.com/devWayne/mBase.css.git
$ cd mBase.css
$ npm install
$ gulp
```
