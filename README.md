# mBase
> mBase.css is used in the movie business department of dianping.com

Powered by Less & Postcss

## Build From Source

```shell
$ git clone https://github.com/devWayne/mBase.css.git
$ cd mBase.css
$ npm install
$ gulp
```

## Example

```shell
$ gulp dist-example
$ open example/index.html

```

## Upload Example

```shell
$ npm run upload
```
