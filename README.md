# ns.css
> Ns css is used in the movie business department of dianping.com

Powered by Less & Postcss

# Usage

**Install with Bower:**

```shell
$ bower install --save ns.css
```

# Build From Source

```shell
$ git clone https://github.com/devWayne/ns.css.git
$ cd ns.css
$ npm install
$ gulp
```
